Tournament Planner Version 1.0 26/04/2016

General usage:
- database to keep track of players and matches in a swiss system tournament
- it allows to:
 register players, delete players, get the lists of players;
 enter matches, get a list of the players and their win records sorted by wins;
 get the matches table with records of winner and loser;
 get a list of pairs of players for the next round of a match.

 Installation requirements:
 - to run a Tournament Planner you neeed to have installed:
   - Vagrant (www.vagrantup.com) which has PostgreSQL installed and configured
   - Virtual Box (www.virtualbox.org)
   - Python (http://python.org/download)
